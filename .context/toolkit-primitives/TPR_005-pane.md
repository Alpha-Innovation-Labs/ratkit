# TPR_005: Pane Primitive

## Desired Outcome

The Pane primitive provides styled panel components with title, icon, padding, and optional footer. Users can render panels with customizable borders, titles, icons, and footers for building terminal UI layouts.

## Next Actions

| Description | Test |
|-------------|------|
| Pane renders with title displayed in top border | `pane_renders_with_title` |
| Pane with icon displays icon before title in border | `pane_with_icon_displays_icon_before_title` |
| Pane applies configurable padding around content | `pane_padding_applied_correctly` |
| Pane displays optional footer text in bottom border | `pane_footer_displays_text` |
| Pane applies custom border style to all borders | `pane_border_style_applied` |
| Pane changes border type when configured | `pane_border_type_changed` |
