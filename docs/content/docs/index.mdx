# Ratatui Toolkit

Ratatui Toolkit is a comprehensive collection of reusable components for the [Ratatui](https://github.com/ratatui-org/ratatui) TUI framework. This library provides pre-built, customizable components to accelerate terminal UI development.

## Features

- **Reusable Components**: Ready-to-use components that work out of the box
- **Type-Safe**: Built with Rust's strong type system
- **Customizable**: Each component can be styled and configured to fit your needs
- **Well-Documented**: Comprehensive documentation with examples

## Components

<Steps>
  <Step title="Layout Components">
    ResizableSplit, MenuBar
  </Step>
  <Step title="UI Components">
    Button, Dialog, Toast, HotkeyFooter
  </Step>
  <Step title="Navigation">
    TreeView, FileSystemTree
  </Step>
  <Step title="Rendering">
    MarkdownRenderer, TermTui, StatusLineStacked
  </Step>
</Steps>
