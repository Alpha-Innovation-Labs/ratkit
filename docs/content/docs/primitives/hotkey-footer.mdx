# HotkeyFooter

The `HotkeyFooter` component displays keyboard shortcuts and hints at the bottom of the screen.

## Features

- Customizable key bindings display
- Multi-line support
- Context-aware shortcuts
- Auto-width adjustment

## Usage

```rust
use ratatui_toolkit::HotkeyFooter;

let footer = HotkeyFooter::new()
    .add_binding("Ctrl+Q", "Quit")
    .add_binding("Ctrl+S", "Save")
    .build();
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `bindings` | `Vec<(String, String)>` | `[]` | List of key-action pairs |
| `separator` | `String` | `"|"` | Separator between bindings |
