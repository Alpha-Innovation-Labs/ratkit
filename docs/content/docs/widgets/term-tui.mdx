# TermTui

The `TermTui` component provides a foundation for terminal UI rendering and management.

## Features

- Terminal abstraction
- Event handling
- Render loop management
- Cross-platform support

## Usage

```rust
use ratatui_toolkit::TermTui;

let mut term = TermTui::new()
    .with_title("My App")
    .start();

term.run(|ui| {
    // Render UI components
});
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `String` | - | Window title |
| `tick_rate` | `Duration` | `250ms` | Refresh rate |
| `fullscreen` | `bool` | `false` | Start in fullscreen |
