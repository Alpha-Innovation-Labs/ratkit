# ResizableSplit

The `ResizableSplit` component allows you to create split pane layouts that users can resize interactively.

## Features

- Horizontal or vertical orientation
- Keyboard-based resizing
- Configurable minimum and maximum sizes
- Smooth and responsive resizing

## Usage

```rust
use ratatui_toolkit::ResizableSplit;

let mut split = ResizableSplit::horizontal()
    .with_min_size(10)
    .with_max_size(50)
    .build();
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `Orientation` | `Horizontal` | Split direction (Horizontal/Vertical) |
| `min_size` | `u16` | `5` | Minimum size for each pane |
| `max_size` | `u16` | `100` | Maximum size for each pane |
| `initial_ratio` | `f32` | `0.5` | Initial size ratio (0.0-1.0) |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+H` | Move splitter left |
| `Ctrl+L` | Move splitter right |
| `Ctrl+J` | Move splitter down |
| `Ctrl+K` | Move splitter up |
