# MenuBar

The `MenuBar` component provides a top navigation menu with keyboard shortcuts and dropdown menus.

## Features

- Hierarchical menu structure
- Keyboard navigation
- Shortcut display
- Menu item highlighting

## Usage

```rust
use ratatui_toolkit::MenuBar;

let menu = MenuBar::new()
    .add_menu("File", vec![
        MenuItem::new("Open", "Ctrl+O"),
        MenuItem::new("Save", "Ctrl+S"),
    ])
    .build();
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `menus` | `Vec<Menu>` | `[]` | List of menus |
| `active_index` | `usize` | `0` | Currently active menu |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Alt+F` | Open File menu |
| `←` / `→` | Navigate menus |
| `↑` / `↓` | Navigate items |
| `Enter` | Select item |
| `Escape` | Close menu |
